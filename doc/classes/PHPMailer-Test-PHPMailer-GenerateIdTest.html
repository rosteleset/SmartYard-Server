public testGenerateID() : mixed

To fully test the method under test, the tests should be run on the following system configurations:

  • PHP >= 7.0 to hit random_bytes();
  • PHP < 7.0 with OpenSSL enabled to hit openssl_random_pseudo_bytes();
  • PHP < 7.0 with the OpenSSL extension disabled;
  • PHP >= 7.0 with the OpenSSL extension disabled and the random_bytes() function in the php.ini disable_functions list.

Note: The exact text string length of result may vary due to the str_replace() in the final statement of the method, but it should always be at least 32 characters long.
