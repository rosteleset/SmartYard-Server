# `server/services/sys_exporter` — Prometheus exporter

Prometheus exporter for IP intercom monitoring.

## Install and run

1. Create `.env`:

```shell
cp .env_example .env
```

2. Dependencies:

```shell
npm i
```

3. Start:

```shell
npm start
```

## Development

Docker:

```shell
docker compose up -d
```
