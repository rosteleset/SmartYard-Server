# `test.php`

Локальная точка входа для разработки/тестирования.

## Назначение

`test.php` предназначен для **локального тестирования** во время разработки.

## Примечания

- Эта точка входа **не является публичным production-интерфейсом**.
- Часто её держат только локально (нередко добавляют в `.gitignore`), чтобы случайно не выкатывать в прод.
