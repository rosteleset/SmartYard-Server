# `server/api/api.php` — базовый класс API

Этот файл определяет базовый класс `api\api`, который используется endpoint-классами в `server/api/**`.

## Namespace и класс

- **Namespace**: `api`
- **Базовый класс**: `api\api`

## Обработчики HTTP-методов

Базовый класс содержит дефолтные обработчики:

- `GET(array $params)`
- `POST(array $params)`
- `PUT(array $params)`
- `DELETE(array $params)`

По умолчанию все они возвращают `ANSWER(false, "badRequest")` и подразумевается, что конкретные endpoint’ы их переопределяют.

## Формат возврата (контракт с диспетчером)

Методы этого класса возвращают **структурированный массив**, в котором кодируется:

- целевой **HTTP-код ответа** (как ключ массива, например `"200"`, `"204"`, `400`, `403`, …)
- **payload** (данные успеха или объект ошибки)
- опционально **метаданные кеширования** (дополнительным элементом массива)

### `ANSWER($result, $answer, $cache)`

`ANSWER()` — удобная обёртка:

- если `$result === false` → возвращает ошибку через `ERROR($answer)`
- иначе → возвращает успех через `SUCCESS($answer, $result, $cache)`

### `SUCCESS($key, $data, $cache)`

Кодирует успешный ответ.

- Если `$data !== false`:
  - если `$key === "__asis__"`: payload возвращается “как есть” под ключом `"200"`.
  - иначе: payload заворачивается как `"200": { "$key": $data }`.
- Если `$data === false`: возвращает `"204" => false`.

Кеширование:

- `$cache` приводится к `int`.
- если `$cache < 0`, используется глобальный `$redis_cache_ttl`.
- значение кеша добавляется отдельным элементом массива: `["cache" => $cache]`.

### `ERROR($error)`

Кодирует ошибку.

- Если `$error` пустой, берётся `getLastError()`, а если и он пуст — `"unknown"`.
- Известные имена ошибок мапятся на HTTP-коды:
  - `badRequest` → 400
  - `forbidden` → 403
  - `notFound` → 404
  - `notAcceptable` → 406
- Неизвестные имена ошибок по умолчанию дают 400.

Форма результата:

- `{ httpCode: { "error": "<errorName>" } }`

### `index()`

`index()` — внутренний хелпер для **индексации методов**. В базовой реализации возвращает `false`.
