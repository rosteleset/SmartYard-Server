# Backend `frs`

## Назначение

Интеграция с сервисом распознавания лиц / событий (FRS): потоки, лица, колбэки — см. константы методов в базовом классе.

## Код

- **Базовый класс**: `server/backends/frs/frs.php`.
- **Варианты**: `internal`.

## Конфигурация

Ключ в `server/config/config.json`: `backends.frs`.

## Основные методы (контракт)

Большой контракт с константами `M_*` / `P_*`; операции над потоками и событиями.

## Кто использует

`plog`, интеграции видеоаналитики.

См. также [общий индекс backend’ов](../README.ru.md) и [загрузчик `loader.php`](../../utils/loader.ru.md).
