# Backend’и для Accounts

В этом разделе описаны backend’и, которые напрямую используются Accounts API и связанными сценариями:

- `users`
- `groups`
- `authentication`
- `authorization`

Для каждого backend’а есть раздел **Зависимости**: конфиг, хранилища/ключи, и кто его вызывает сверху (API endpoint’ы/UI).
